Electric poles Ended
Description

There are N electric poles on road, with some energy A[i]. Pikachu gets 5 energy if he passes an electric pole which has energy greater than the previous pole. You are allowed to change the position of poles so that Pikachu gets the maximum energy.

Find the maximum energy Pikachu can get.


Input
Input Format

First-line contains T no of test cases.

The first line of each test case contains N, no of electric poles.

The second line of each test case contains N spaced integers Energy of electric poles.

Constraints

1 <= T <= 1000

1 <= N <= 1000

1 <= A[i] <= 1000


Output
Output Format

For each test case output, the maximum energy Pikachu can get on the new line.


Sample Input 1 

4
3
1 2 3
5
1 2 3 4 5
4
1 1 1 1
5
1 1 1 2 2
Sample Output 1

10
20
0
10
